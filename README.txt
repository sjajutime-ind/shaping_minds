Shaping Minds Website Package

Files included:
- index.html
- assets/css/style.css
- assets/js/main.js
- assets/img/*.svg

How to use:
1. Upload the full folder to your hosting/server.
2. Open index.html in browser or set it as your homepage.
3. Replace the placeholder contact details, address, and any text as needed.
4. To connect the enquiry form to backend/Google Sheets/WhatsApp, add your form action or JS integration inside assets/js/main.js.

Notes:
- This package uses local SVG image assets, so it works without depending on external image URLs.
- Google Fonts are loaded online. If you want, they can later be replaced with local fonts too.
